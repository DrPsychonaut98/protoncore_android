HumanVerificationRemoteRepository

interface HumanVerificationRemoteRepository : Invokable

Remote repository interface that defines all operations that the dta layer (module) should implement. All of these operations are network (remote) dependant and often it is required to have internet connection to execute successfully.

It is required to run them only on a IO thread.
