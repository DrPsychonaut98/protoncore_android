ExecutorsTest

interface ExecutorsTest

An interface meant to be implemented by a Test Suite that uses Android code that internally uses Executors. Example:

class MyClassTest : ExecutorsTest by executorsTest {
// test cases
}

It provides a ExecutorsTestRule and alternative dispatchers.
