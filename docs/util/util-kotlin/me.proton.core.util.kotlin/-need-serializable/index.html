NeedSerializable

@Target([AnnotationTarget.FUNCTION]) annotation class NeedSerializable

Elements marked with this annotation are expecting to have a parameter which is annotated with Serializable annotation; an exception will be thrown otherwise.

Target has been set to be AnnotationTarget.FUNCTION because AnnotationTarget.TYPE_PARAMETER is not supported yet, but in future we are expecting targets to be AnnotationTarget.TYPE_PARAMETER and AnnotationTarget.VALUE_PARAMETER
