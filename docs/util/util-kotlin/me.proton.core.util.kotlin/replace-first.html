replaceFirst

inline fun <C : MutableCollection<T>, T> C.replaceFirst(newItem: T, predicate: (T) -> Boolean): Boolean

Replace first the element matching the predicate with newItem. newItem will be added even if no item matches the predicate will be added at the end of the collection, if it is ordered

Return
true if element has been replaced or simply added successfully

Receiver
C is MutableCollection of T
