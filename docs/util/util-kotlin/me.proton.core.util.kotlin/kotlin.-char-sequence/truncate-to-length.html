truncateToLength

fun CharSequence.truncateToLength(maxLength: Int): CharSequence

Return
CharSequence. If receiver CharSequence is shorted than the given maxLength, return itself. Else CharSequence.substring from 0 to maxLength and add "..." at the end of it. E.g. > val hello = "Hello world!" println( hello.truncateToLength( 9 ) ) // Hello Wo...
