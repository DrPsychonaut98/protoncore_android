UiModelMapper

interface UiModelMapper<in Ein, UI, out Eout>

A common interface for transform an Entity to an UiModel and transform back an UiModel to an entity.

Parameters

Ein - the type of the source entity to map to UiModel UI

UI - the type of the UiModel

Eout - the type of the entity generated from UiModel UI
