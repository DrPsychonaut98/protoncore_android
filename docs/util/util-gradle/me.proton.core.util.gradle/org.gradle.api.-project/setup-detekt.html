setupDetekt

fun Project.setupDetekt(filter: (Project) -> Boolean = { true }): Unit

Setup Detekt for whole Project. It will:

  • apply Detekt plugin to sub-projects
  • configure Detekt Extension
  • add accessor Detekt dependencies
  • register MergeDetektReports Task, in order to generate an unique json report for all the module

Parameters

filter - filter Project.subprojects to attach Detekt to
