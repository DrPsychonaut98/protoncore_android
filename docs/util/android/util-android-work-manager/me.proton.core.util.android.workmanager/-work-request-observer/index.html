WorkRequestObserver

class WorkRequestObserver

An entity that allow us to observe a WorkRequest in a nice concise style. It will be generate by WorkRequest.observe extensions, and allow us to set callback like the following:

  • onSuccess with a Data
  • onSuccess with a deserialized object
  • onFailure
  • onStateChange with a WorkInfo.State
