WorkManagerActivity

interface WorkManagerActivity : LifecycleOwner

An interface containing Lifecycle-driven observe extensions functions for Activity

Implements this interface on your Activity for be able to call WorkRequest extensions without passing the LifecycleOwner explicitly. I.e. workRequest.observe{ ... } it's a valid function that will observe the data respecting the Activitys Lifecycle.

Refer to ObserverExtensions.kt for methods documentation. Functions can't be inlined since it's inside an interface and cannot be extensions since they're already extensions on WorkRequest
