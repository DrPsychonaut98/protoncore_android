Instantiates ApiManager for given Api interface and user.

Parameters

Api - Retrofit interface defined by the client, must inherit from BaseRetrofitApi.

baseUrl - Base url for the api e.g. "https://api.protonvpn.ch/"

userData - UserData to be used in the ApiManager.

interfaceClass - Kotlin class for Api interface.

networkManager - NetworkManager to be used for connectivity checks.

clientErrorHandlers - Extra error handlers provided by the client.

certificatePins - Overrides Constants.DEFAULT_PINS

Return
ApiManager instance.
