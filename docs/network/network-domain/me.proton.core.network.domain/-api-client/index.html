ApiClient

interface ApiClient

Represents the client of the library. Enables 2-way communication between the lib and the client.

Properties

appVersionHeader

Client's value for 'x-pm-appversion' header.

abstract val appVersionHeader: String

backoffBaseDelayMs

Base value (in milliseconds) for exponential backoff logic. e.g. for value 500 first retry will happen randomly after 500-1000ms, next one after 1000-2000ms ...

open val backoffBaseDelayMs: Int

backoffRetryCount

Retry count for exponential backoff.

open val backoffRetryCount: Int

dohTimeoutMs

Global timeout for DoH logic.

open val dohTimeoutMs: Long

enableDebugLogging

Enables debug logging in the underlying HTTP library.

abstract val enableDebugLogging: Boolean

shouldUseDoh

Tells the lib if DoH should be used in a given moment (based e.g. on user setting or whether VPN connection is active). Will be checked before each API call.

abstract val shouldUseDoh: Boolean

timeoutSeconds

Timeout for internal api call attempt (due to error handling logic there might be internal calls in a single API call by the client.

open val timeoutSeconds: Long

userAgent

Client's value for 'User-Agent' header.

abstract val userAgent: String
