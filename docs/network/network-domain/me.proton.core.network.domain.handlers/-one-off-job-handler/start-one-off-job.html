startOneOffJob

suspend fun startOneOffJob(inputParameter: IN, block: suspend (IN) -> OUT): OUT

Starts one-off job. Will prevent multiple time calls to start more than one parallel job. So in other words makes sure not to start additional instances if the job is running.

Parameters

inputParameter - the parameter needed for the Job.

block - higher-order function representing the Job that should be executed.
