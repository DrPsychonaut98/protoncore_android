unfilteredList

abstract var unfilteredList: ModelsList

A reference to all the items without any filter applied. This must be be updated in sub-classes, every time an update for items is received. e.g. override androidx.recyclerview.widget.ListAdapter.submitList for Adapters that inherit from androidx.recyclerview.widget.ListAdapter
