ProtonAdapter

abstract class ProtonAdapter<UiModel, ViewRef : Any, ViewHolder : ViewHolder<UiModel, ViewRef>> : ListAdapter<UiModel, ViewHolder>, ClickableAdapter<UiModel>, FilterableAdapter<UiModel, List<UiModel>>

A ListAdapter for RecyclerView that contains a List of UiModel items. Implements ClickableAdapter

A basic implementations expects

  • an implementation of DiffUtil.ItemCallback of UiModel
  • an implementation of ClickableAdapter.ViewHolder that will be passes a generic UiModel and will be created by overriding onCreateViewHolder

Parameters

onItemClick - A lambda with UiModel as parameter that will be triggered when an item is clicked

onItemLongClick - A lambda with UiModel as parameter that will be triggered when an item is long clicked

diffCallback - a REQUIRED implementation on DiffUtil.ItemCallback of UiModel that will be used for compare the items.
